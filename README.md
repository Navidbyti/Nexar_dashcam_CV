# پروژه تشخیص حوادث رانندگی با ویدیو (Driver Accident Detection)

این پروژه مربوط به **تشخیص نقاط برخورد و حوادث رانندگی در ویدیوها** با استفاده از شبکه‌های عصبی کانولوشنی سه‌بعدی (3D CNN) و پردازش فریم‌های ویدیویی است.

---

## توضیحات کلی

- **هدف:** شناسایی لحظه برخورد یا حادثه در ویدیوهای رانندگی.  
- **داده‌ها:** ویدیوهای ضبط شده از دوربین داشبورد خودرو با طول‌های مختلف.  
- **مدل:** 3D CNN که فریم‌های ویدیویی را به صورت دسته‌ای (chunks) پردازش می‌کند.  
- **ویژگی‌ها:**  
  - پیش‌پردازش ویدیوها با برش مربع و تغییر اندازه به 224×224.  
  - استخراج 9 فریم مهم اطراف نقطه حادثه یا مرکز ویدیو.  
  - آموزش مدل به صورت تدریجی روی chunks برای کاهش مصرف حافظه و افزایش پایداری.  
- **متریک‌ها:** دقت (Accuracy) روی مجموعه آموزش، اعتبارسنجی و تست.
